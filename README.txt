ミューオンの崩壊曲線と、ラーモア歳差運動のフィットをする
.datファイルを読み込んで、ヒストグラムとそのフィット曲線を描画する


実行方法
VScodeでroot1フォルダを開く
ターミナルで、
root
(ROOTが起動)
.x graph_fitting1.cpp
(fitting1.pngというファイルが作られる)
.x graph_fitting1_larmor.cpp
(fitting1_larmor.pngというファイルと、fitting.txtというファイルが作られる)
ROOTから抜けるときは.q


graph_fitting1.cppが崩壊曲線
graph_fitting1_larmor.cppがラーモア歳差運動のフィット


.datファイルの中身(たぶん)
1列目：下シンチの信号の時刻(ミューオンの崩壊時刻)
2列目：上シンチの信号の時刻(ミューオンの入射時刻)
3列目：ミューオンの崩壊時間(1列目ー2列目)


.datファイルとは？
.datは「data（データ）」の略。
特定の汎用的なフォーマットを持たないデータ保存用ファイル。
拡張子が.datというだけで、「中身の形式」は決まっていない。
中身はテキスト（数字や文字の羅列）のこともあるし、バイナリ形式（人間には読めないコンピュータ用の形式）のこともある。
